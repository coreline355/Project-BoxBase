# Home

**Bringing TV Box info to a Unified Place.**

BoxBase is a Tv Box Wiki project that aims to unify obscure TV Box information in one place. Eliminating the necessity to hunt for obscure information in the World Wide Web.

!!! warning

    Project BoxBase is a non-profit project that aims to unify TV Box information. 
    This project does not help:
    
    - Installing forbidden IPTV clients
    - Praticing and/or doing any form of Piracy

## [Click here to see documented Devices](devices.md)

## Project BoxBase needs your help!

BoxBase is a very new wiki. With a growing index of documented devices. Please help grow the number of documented devices by documenting any TV Box that you own and add a pull request to the [BoxBase Github](https://github.com/coreline355/Project-BoxBase).
